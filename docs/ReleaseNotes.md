# v1.5 Release notes - What's new

## Overview:
- Update task & project assign commands to accept task_index or task_name & project_index or project_name

## Person related commands
- `COMMAND` - SHORT COMMAND DESCRIPTION

## Task related commands
- `task assign PERSON_INDEX (-n TASK_NAME | -i TASK_INDEX)`<br> e.g., `task assign 1 -n Draft proposal` or `task assign 1 -i 2` - now accepts TASK_NAME or TASK_INDEX

## Project related commands
- `project assign PERSON_INDEX (-n PROJECT_NAME | -i PROJECT_INDEX)`<br> e.g., `project assign 1 -n WebApp` or `project assign 1 -i 2` - now accepts PROJECT_NAME or PROJECT_INDEX
